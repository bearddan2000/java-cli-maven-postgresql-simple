# java-cli-maven-postgresql-simple

## Description
A java maven build, that connects to postgresql database.

## Tech stack
- java
- maven
  - 6.8.2

## Docker stack
- postgresql:alpine
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
